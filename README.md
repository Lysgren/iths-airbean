# Airbean

## Beskrivning
Ni ska bygga en mobilanpassad liten webbshop för drönarlevererat kaffe (drönare ingår ej).

## Specifikation

* Splash screen
* Navigation i hamburgermeny
* Kundkorg
* Mockat API, (se api-mappen för boilerplate)
* Produktlista
* Enkel registrering med validering
* Orderhistorik
* En aboutsida
